// Las propiedades de un objeto son los elementos en los que almacenaremos valores en
// el, estas tienen un nombre unico que no se podra repetir en el mismo objeto.

// Los metodos nos sirven para realizar una funcion o accion dentro de un objeto, estas
// se escriben como cualquier funcion, dentro de una propiedad.

// Los bucles for... in son como cualquier otro tipo de bucle, pero este nos permite recorrer todas las
// propiedades de un objeto como tambien los valores asociados a este. Se escriben un poco distinto que
// un bucle for convencional.
